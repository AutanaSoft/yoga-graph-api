# Language Guidelines

- Siempre responde en Español cuando te comuniques con el usuario.
- Siempre responde en Español para toda la documentación técnica.
- Usa inglés para los nombres de variables, configuraciones, nombres de archivos y directorios.
- Nunca realices un commit o push a menos que el usuario lo solicite explícitamente.
- usa pnpm para gestionar las dependencias del proyecto.
